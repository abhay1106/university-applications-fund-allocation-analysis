# 🎓 University Applications & Fund Allocation Analysis

## 📌 Project Overview

This project focuses on analyzing university applications and fund allocation data using Power BI. The objective is to identify application trends, evaluate university performance, assess the impact of special grants, and compare funding distribution across institutions.

The project demonstrates an end-to-end Business Intelligence workflow, including data transformation, data modeling, DAX calculations, and interactive dashboard development.

---

## 🎯 Business Objective

* Analyze university application trends across institutions and states.
* Evaluate funding allocation patterns among universities.
* Identify top and bottom performing universities based on applications.
* Assess the impact of special grants on application volumes.
* Generate actionable insights through interactive Power BI dashboards.

---

## 📂 Dataset Information

### Dataset 1: University Admissions_New

* Records: 4,000 Rows
* Attributes: 23 Columns

### Dataset 2: University Applications Charter

* Records: 900 Rows
* Attributes: 22 Columns

The datasets contain information related to university applications, applicant demographics, institutional details, funding allocations, and special grant programs.

---

## 🛠 Tools & Technologies

* Microsoft Excel
* Power BI
* Power Query
* DAX (Data Analysis Expressions)

---

## 🔄 Data Transformation (Power Query)

The following Power Query transformations were applied:

* Append Queries
* Merge Queries
* Split Columns
* Fill Operations
* Conditional Columns
* Index Columns
* Pivot & Unpivot Transformations
* Data Formatting
* Summary Table Creation

These transformations improved data quality and prepared the datasets for analytical reporting.

---

## 📊 Data Modeling & DAX

* Established relationships between transformed datasets.
* Created custom DAX measures for analytical calculations and dashboard reporting.
* Developed a structured data model to support interactive visualizations.
